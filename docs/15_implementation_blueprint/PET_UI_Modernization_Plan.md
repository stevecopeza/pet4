# PET UI Modernization — Master Implementation Plan

## Objective
Standardize all admin UI surfaces to Level 3 control surfaces:
- Summary band
- Control surface
- Semantic rows
- High signal density

## Constraints
- No domain changes
- No workflow changes
- No API changes (unless explicit)
- Additive only

## Phases
### Phase 1 (Core)
- Projects
- Finance
- Quotes / Leads

### Phase 2 (Context)
- Customers
- Employees / Teams

### Phase 3 (Supporting)
- Support (legacy)
- Advisory
- Escalations
- Knowledge / Conversations / Approvals / Activity

### Phase 4 (Low priority)
- Roles
- Settings
- Tools

## Execution Order
1. Projects
2. Finance
3. Quotes / Leads
4. Customers
5. Employees / Teams
6. Others

## Success Criteria
- All surfaces understandable in <3 seconds
- No WordPress-like UI remains
- No raw tables without context
