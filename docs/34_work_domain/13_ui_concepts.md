## Rules

-   No nested horizontal tabs.
-   Vertical sub-navigation only.
-   Dashboards are read-only.
-   Publishing Role locks version.
-   Assignment screens enforce certification compliance.
-   Compliance violations visibly flagged.
-   Hard validation on mandatory fields.
