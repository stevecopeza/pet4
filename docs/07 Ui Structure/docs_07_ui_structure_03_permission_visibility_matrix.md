# PET – Permission Visibility Matrix

## Purpose of this Document
This document maps **roles and organisational structure** to **menu visibility and screen access**.

Permissions are structural, not discretionary.

---

## Role Categories

- Executive
- Manager
- Project Manager
- Sales
- Engineer
- Support
- Admin

## Managerial Inheritance

Managers automatically:
- See all subordinate team data
- See aggregated KPIs

No per-object sharing exists.

---

## Sensitive Data Rules

- Pay, compensation, and performance notes are Admin-only
- Team members never see peer-sensitive data
